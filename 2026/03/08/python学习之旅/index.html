入门

python作为编写agent的基础，随着AI的发展，传统的码农将会走向没落，所以才开了这篇python学习的帖子，说不定哪天这帖子就写一些agent开发，当然以后还会有一些mcp的建立，ai是个好工具（被ai大人吓哭了），传统的写代码会被AI取代，只有在维度上胜过AI才可以，当然练习与实战是必不可少的（理论结合实战才能学好代码）
